Security Practices

Version 1.0 — 2026

At Toledah Technologies Ltd (“Toledah”, “we”, “us”), the security of your business data is our top priority. Zaena ERP is designed to protect your data while using modern cloud infrastructure provided by Supabase.

1. Data Storage & Database Security

  • Each customer has a dedicated database, isolated from others.
  • Supabase automatically handles database security, updates, and patches.
  • Your data is never shared with other customers.

2. Encryption

  • All data in transit is encrypted with HTTPS/TLS.
  • All data at rest is encrypted with AES-256.
  • Backups stored by Supabase are also encrypted.

3. Backups & Disaster Recovery

  • Supabase maintains daily automated backups of your database.
  • You can manually export or restore your database anytime from the dashboard.
  • Supabase ensures your data can be recovered in case of unexpected events.

4. Authentication & Access Control

  • User accounts are managed via Supabase Auth, which supports secure login methods including passwords, OAuth, and magic links.
  • Passwords are stored securely with bcrypt hashing.
  • You can enforce Row-Level Security (RLS) to control exactly which users can access which data.

5. Monitoring & Logging

  • Supabase tracks database activity, errors, and system metrics automatically.
  • We monitor performance and uptime to ensure reliable service.

6. Network & Infrastructure Security

  • Zaena ERP runs on Supabase cloud infrastructure, hosted on trusted providers (AWS, GCP, or Azure).
  • Servers are monitored, patched, and protected from network attacks.
  • Physical security, firewalls, and DDoS mitigation are handled by the cloud provider.

7. What We Require From You

  • Use strong passwords and keep your login credentials private.
  • Follow best practices for API keys and secrets.
  • Implement proper access control rules in your account and tables.

8. Responsible Disclosure

  • If you discover a security vulnerability, please contact us at:
  • Email: support@toledah.com
  • We treat all reports with high priority and resolve issues responsibly.
